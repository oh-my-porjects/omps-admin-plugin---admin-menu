# 菜单元数据 文档索引

这里是 `admin-menu` 模块的事实源入口。AI 修改本模块前应先读本页，再读 `module-contract.yaml`、`api-docs/`、`tests/` 和相关源码。

## 必读事实

- [模块事实](./模块事实.md)：模块职责、数据表、接口、实时事件和测试资产边界。
- `../module-contract.yaml`：稳定模块契约，包含表归属、生命周期和实时事件声明。
- `../api-docs/`：HTTP API、字段结构、endpoint_key 和接口规则。
- `../tests/README.md`、`../tests/data.yaml`、`../tests/*.test.json`：自动测试资产和数据准备声明。
- `../plugin.yaml`、`../admin-web.yaml`：模块元数据、环境变量和后台页面种子。

## 当前结论

- 本仓库是 `菜单元数据` 的真实模块仓库，不是 `module-template`。
- 旧 `module-index.json` 不再作为仓库事实源；派生索引由平台生成到工作区 `.gitNexus/module-indexes/admin-menu.json`。
- 修改接口、字段、表、推送事件或测试数据时，必须同步更新本页列出的事实源。
